Extractor¶

Extractor.ra_dec_format(val)¶

Ra/Dec string formatting

Converts the input string format of a right ascension/ declination coordinate to one recognizable by astroquery

Parameters

val (str) – string; an ra/dec expression formatted as “005313.81 +130955.0”.

Returns

the ra/dec coordinates re-formatted as “00h53m13.81s +13d09m55.0s”

Return type

string
